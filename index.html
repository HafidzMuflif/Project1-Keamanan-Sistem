Block Cipher Encryption Tool

Enkripsi dan Dekripsi Teks menggunakan Mode Cipher Block

Catatan Penggunaan

  • Key / Kunci dapat berupa kombinasi huruf atau angka, contoh: kunci123
  • Initialization Vector (IV) digunakan pada mode CBC, OFB, dan CFB.
  • IV dapat berupa teks bebas seperti contoh: init atau 1234
  • Mode ECB tidak memerlukan IV.
  • Sistem ini menggunakan block size 4 karakter untuk memproses teks.
  • Jika teks tidak habis dibagi 4, maka sistem akan menambahkan spasi secara otomatis.
